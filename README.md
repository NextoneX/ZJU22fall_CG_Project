## 工程管理

使用VS来管理工程，需要做如下设置：
设置包含目录为该文件夹下的include
设置库目录为该文件夹下的lib
设置连接器的输入为glfw3.lib
同时需要包含src中的cpp文件(sphere实际上没用到)
